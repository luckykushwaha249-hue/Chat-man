# chat-man (Offline Bluetooth Chat)

A fully offline, real-time Bluetooth desktop chat application built with Python and Flet.

## Features
* **Lock Screen Security:** Secured with a predefined password.
* **100% Offline:** Uses Bluetooth (RFCOMM/Sockets) for communication.
* **Real-time Chat:** Instant messaging, typing indicators, auto-scroll.
* **Auto-Disconnect Detection:** 10-second heartbeat monitor triggers pop-ups on drop.
